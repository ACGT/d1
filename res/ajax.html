lice is an innovative programming environment that makes it easy to create animation for telling a story, playing an interactive game, or a sharing a video on the web. Alice uses 3D graphics and a drag-and-drop interface to make the learning experience fun and engaging. Designed to be a student's first exposure to object-oriented programming, it is a fun and easy way to begin learning the Java programming language. The Alice teaching tool was created by Carnegie Mellon University and is available for free at alice.org!
